What is Liberatube?
Liberatube is a privacy, feature rich alternative front end to YouTube that uses the PHP scripting languange as the backend.

Installation

To install Liberatube, it is necessary to have a program capable of running PHP scripts or use a hosting provider that supports PHP. We recommend utilizing Apache2 in conjunction with PHP for local installations. Alternatively, for external hosting, InfinityFree is a recommended choice due to its free availability. In addition, a SQL-capable database program is required. MySQL is a recommended option, as most hosting providers supporting PHP also support MySQL.

Once the necessary prerequisites are in place, follow these steps to install Liberatube:

1. Download the desired version of Liberatube from the official GitHub repository.
2. Extract the contents of the downloaded package to the root directory of your website or document root.
3. Open the "config.php" file and proceed to configure the database details. This includes providing the necessary credentials for connecting to the database.
4. Customize the configuration to specify user access privileges for the admin dashboard, if desired.
5. Optionally, configure different instances of Invidious according to your requirements.

By following these steps, you can ensure a professional installation of Liberatube, enabling you to enjoy its features and functionality.

Privacy

Liberatube prioritises user privacy by abstaining from utilizing any official Google APIs. Instead, it employs the Invidious API to ensure a private and secure experience for all users. By relying on the Invidious API, Liberatube avoids potential data collection and tracking associated with Google's APIs, reinforcing the commitment to user privacy and data confidentiality.

Accounts

The account system uses 1 SQL database to store username and passwords. The passwords are stored in the database using next generation storage technology using SHA-512 hashing and 8 character before and after salts.

About

Liberatube is a free and open-source, Feature rich alternative front end to YouTube, designed to offer users a better and modern user interface while ensuring a privacy-focused experience with no trackers or ads. It provides a streamlined interface to access and view YouTube videos while prioritizing user privacy and freedom from intrusive elements commonly found on mainstream video platforms.
